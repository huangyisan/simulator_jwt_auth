# simulator_jwt_auth
模拟jwt登陆认证

## 一句话介绍
后端python作为登陆验证, 前端vue编写登陆界面, 实现jwt方式认证登陆, 主要是自己学习jwt, 所以写着玩的.

## 环境
1. python v3.6.4
2. node v12.15.0

## 使用
1. `cd backend; pip install -r requirements.txt; python webServer.py`
2. `cd backend/ordersys ; npm i ; npm run serve`
